### 注：JQuery练手，纯属一乐

#### 功能

将谷歌浏览器导出的`HTML`格式书签转为`JSON`格式作为数据源生成自定义的导航网站

#### 说明

- 转换后的json数据保存在线上（如七牛云）通过`ajax`获取
- 本代码利用~~逐渐被淘汰的~~`jQuery`读取筛选数据并生成`HTML`单页面
